## Contribution

To get started with the Data Engineering Staffing Helper, follow these steps:

1. **Clone the repository**:
    ```sh
    git clone /home/dany/workspace/gen-ai-sprint
    cd gen-ai-sprint
    ```

2. **Install dependencies**:
    ```sh
    pip install -r requirements.txt
    ```

3. **Set up environment variables**:
    - Get your [NVIDIA genAI](c) key.
    - Create a `.env` file at the root of the repository and add your key:
        ```
        NVIDIA_GENAI_KEY=your_key_here
        ```

4. **Run the application**:
    ```sh
    python main.py
    ```

You are now ready to use the Data Engineering Staffing Helper!
